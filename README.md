## ğŸ§  How It Works

1. Resume content is stored in the database.
2. User submits a question through the chat interface.
3. Frontend sends the question to the FastAPI backend.
4. Backend constructs a structured prompt.
5. OpenRouter processes the prompt using the selected AI model.
6. AI generates a response strictly based on resume data.
7. Response is returned to frontend and displayed in the chat UI.
